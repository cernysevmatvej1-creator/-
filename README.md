Group — мобильное приложение (клиент)
Group — мобильное приложение на .NET MAUI, клиент для GroupApi. Реализует работу с группами, поиск, создание новых заявок и просмотр существующих. Использует MVVM, сервисы, репозитории и интерфейсы для чистой архитектуры. Дальше планнируется добавление людей и чата 

Технологии: C#, .NET MAUI, XAML, MVVM, HTTP-запросы к API.

Статус: Активно разрабатывается. В планах — улучшение интерфейса и расширение функционала.
